# Smart Tab Companion for Whale Browser

A smart sidebar extension that provides contextual summaries, quick notes, and page references.

## Features
- Detects the current tab and shows contextual info.
- Allows quick note-taking linked to each page.
- Stores notes locally for easy reference.

## How to Install
1. Go to [NAVER Whale Developer Center](https://developers.whale.naver.com/).
2. Compress this folder into `SmartTabCompanion.zip`.
3. Upload the ZIP as a new “Sidebar Extension”.
